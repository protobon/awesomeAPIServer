# Awesome API Server

## Build Project
</br>

```console
$ make build
$ make build stage=local
```
Production: 'make build stage=prod'
#

## Run Project
</br>

### Local
```console
$ make run
```

### Compose Test
```console
$ make up
```

### Detached 
```console
$ make dup
```
Production: 'make dup stage=prod'
#

## Docker Logs (view)
```console
$ make logs
```
Production: 'make logs stage=prod'
